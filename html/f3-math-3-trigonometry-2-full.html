Obtuse Angle Rules

  • sin(180° − θ) = sin θ
  • cos(180° − θ) = −cos θ
  • tan(180° − θ) = −tan θ

Example: sin 150° = sin(180°−30°) = sin 30° = ½
cos 120° = −cos 60° = −½

The Sine Rule

Sine Rule Formula

a/sin A = b/sin B = c/sin C

Where a, b, c are sides opposite to angles A, B, C respectively.

When to Use

  • Given 2 angles and 1 side (AAS or ASA)
  • Given 2 sides and a non-included angle (SSA)

📝 Worked Example

In triangle ABC, A = 50°, B = 70°, a = 8cm. Find b.

C = 180° − 50° − 70° = 60°

b/sin B = a/sin A

b/sin 70° = 8/sin 50°

b = 8 × sin 70°/sin 50°

b = 8 × 0.9397/0.7660

b ≈ 9.81 cm

The Cosine Rule

Cosine Rule Formulas

a² = b² + c² − 2bc cos A
cos A = (b² + c² − a²) / 2bc

When to Use

  • Given 3 sides (SSS) — find an angle
  • Given 2 sides and included angle (SAS) — find the third side

📝 Worked Example

Triangle with b = 7cm, c = 9cm, A = 65°. Find a.

a² = b² + c² − 2bc cos A

a² = 49 + 81 − 2(7)(9) cos 65°

a² = 130 − 126(0.4226)

a² = 130 − 53.25 = 76.75

a ≈ 8.76 cm

Area of a Triangle

Trigonometric Area Formula

Area = ½ ab sin C

Where a and b are two sides and C is the included angle between them.

📝 Worked Examples

Example 1: a = 6cm, b = 8cm, C = 40°

Area = ½ × 6 × 8 × sin 40°

Area = 24 × 0.6428

Area ≈ 15.43 cm²


Example 2: Find angle C if Area = 30cm², a = 10cm, b = 8cm

30 = ½ × 10 × 8 × sin C

sin C = 30/40 = 0.75

C = 48.59° or 131.41°

Angles of Elevation & Depression

Definitions

Angle of Elevation

The angle measured upward from the horizontal to the line of sight to an object above.

Angle of Depression

The angle measured downward from the horizontal to the line of sight to an object below.

📝 Example: Angle of Elevation

A boy 1.6m tall stands 30m from a tree. He looks up at an angle of 40° to the top. Find the height of the tree.

Let h = height of tree above boy's eye level

tan 40° = h/30

h = 30 × tan 40° = 30 × 0.8391 = 25.17m

Total height = 25.17 + 1.6 ≈ 26.8m

Key: Angle of elevation = angle of depression (alternate angles between parallel horizontal lines)

3D Trigonometry

Approach to 3D Problems

  • Identify the relevant 2D triangle within the 3D shape
  • Extract that triangle and work on it separately
  • Use Pythagoras theorem and trig ratios
  • Combine results carefully

📝 Example: Cuboid Diagonal

A cuboid 8cm × 6cm × 5cm. Find the angle the main diagonal makes with the base.

Step 1: Base diagonal = √(8² + 6²) = √100 = 10cm

Step 2: Main diagonal = √(10² + 5²) = √125 = 5√5 cm

Step 3: tan θ = 5/10 = 0.5

θ = 26.57°

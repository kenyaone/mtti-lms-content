Position Vectors & Column Notation

The position vector OP goes from origin O to point P. In 3D, P(x,y,z) has OP = xi + yj + zk.

2D: OP = [x; y]    3D: OP = [x; y; z]
i=(1,0,0)   j=(0,1,0)   k=(0,0,1)

Magnitude and Unit Vectors

|v| = √(x² + y² + z²)
Unit vector: v̂ = v / |v|    (|v̂| = 1)
Example: v = (6, −2, 3). Find |v| and unit vector.
|v| = √(36+4+9) = √49 = 7
v̂ = (6/7, −2/7, 3/7)

Parallel Vectors and Collinearity

Parallel: b = k·a (one is scalar multiple of other)
Collinear A,B,C: AB = k·BC for some k
Example 1: a=(2,4,−6), b=(1,2,−3). Are they parallel?
b = ½a (each component ×½) → Yes, parallel (k=½)
Example 2: A(1,2), B(3,6), C(5,10). Collinear?
AB=(2,4); BC=(2,4) → AB=1·BC → Collinear ✔
Example 3: OA=2i+j, OB=5i+4j, OC=8i+7j. Show A,B,C collinear.
AB=3i+3j; AC=6i+6j=2·AB → parallel and share A → Collinear

Section Formula

P divides AB internally in ratio m:n:

OP = (n·OA + m·OB) / (m + n)
Midpoint (m=n=1): OM = (OA + OB) / 2
Example: A(2,1), B(8,7). P divides AB in ratio 2:1.
OP = (1×(2,1) + 2×(8,7)) / 3 = (2+16, 1+14)/3 = (18,15)/3 = (6,5)

Vector Geometry — Proofs

Midpoint theorem (vectors): OA=a, OB=b. M midpoint of AB.
AB = b − a; AM = ½AB = ½(b−a)
OM = OA + AM = a + ½(b−a) = ½(a+b)
Proportional division: OA=a, OB=b. N divides AB in 1:3.
ON = (3·OA + 1·OB) / 4 = (3a+b)/4
General strategy for vector proofs:
  1. Express all vectors in terms of 2 base vectors (usually OA=a, OB=b)
  2. Find AB, then use scalar multiples
  3. To prove collinear: show AB = k·BC
  4. To find a point: use section formula

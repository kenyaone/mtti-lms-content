Logarithms — Quick Review

Relationship: Logs & Indices

log_a(x) = n  ⟺  aⁿ = x

Read: "log base a of x equals n"

Examples

Logarithm FormIndex Form
log₂8 = 32³ = 8
log₁₀100 = 210² = 100
log₃81 = 43⁴ = 81
log₅(1/25) = −25⁻² = 1/25
log_a(1) = 0a⁰ = 1
log_a(a) = 1a¹ = a
Common log: log₁₀x is written as log x (base 10 assumed). Natural log: logₑx is written as ln x.

Laws of Logarithms

1. Product Law

log(AB) = log A + log B

Example:
log 6 = log(2×3) = log 2 + log 3
= 0.3010 + 0.4771 = 0.7781

2. Quotient Law

log(A/B) = log A − log B

Example:
log(5/2) = log 5 − log 2
= 0.6990 − 0.3010 = 0.3980

3. Power Law

log(Aⁿ) = n log A

Example:
log(8) = log(2³) = 3 log 2
= 3 × 0.3010 = 0.9030

⚠ Common Mistakes

  • log(A+B) ≠ log A + log B
  • log(A−B) ≠ log A − log B
  • (log A)(log B) ≠ log(AB)
Special values: log 1 = 0, log 10 = 1, log 100 = 2, log 0.1 = −1

2. log 500 − log 5 = log(500/5) = log 100 = 2

3. 2log 3 + log 4 = log 9 + log 4 = log 36 ≈ 1.556

4. log 2 + 3log 5 = log 2 + log 125 = log 250

= 2(0.3010) + 0.4771 = 1.0791

Find log 0.5: log(1/2) = log 1 − log 2 = 0 − 0.3010 = −0.3010

Change of Base Formula

Formula

log_a(x) = log_b(x) / log_b(a)

Most commonly: log_a(x) = log(x)/log(a) [using base 10]

Examples

1. log₂(32) = log 32/log 2 = 1.5051/0.3010 = 5 ✓

2. log₃(50) = log 50/log 3 = 1.6990/0.4771 ≈ 3.561

3. log₅(100) = log 100/log 5 = 2/0.6990 ≈ 2.861

Useful Identity

log_a(b) × log_b(a) = 1

This follows directly from the change of base formula.

Solving Logarithmic Equations

Key Principle

If log_a(X) = log_a(Y), then X = Y

Type 1: Simple Log Equations

1. log₃x = 4 → x = 3⁴ = 81

2. log x = 2 → x = 10² = 100

3. log(x+3) = 1 → x+3 = 10 → x = 7

Type 2: Using Log Laws

1. log x + log 5 = 2

log 5x = 2 → 5x = 100 → x = 20

2. log(x+1) − log(x−2) = 1

log[(x+1)/(x−2)] = 1 → (x+1)/(x−2) = 10

x+1 = 10x−20 → 9x = 21 → x = 7/3

Solving Exponential Equations

Method: Take logs of both sides

When the unknown is in the exponent, take log of both sides and use the power law.

Examples

1. 2ˣ = 50

log 2ˣ = log 50 → x log 2 = log 50

x = log 50/log 2 = 1.6990/0.3010 = 5.644

2. 3ˣ = 20

x log 3 = log 20 → x = log 20/log 3

x = 1.3010/0.4771 ≈ 2.727

3. 5²ˣ⁻¹ = 125

5²ˣ⁻¹ = 5³ → 2x−1 = 3 → x = 2

Shortcut: If both sides can be written as the same base, equate the exponents directly.

Natural Logarithms & the Number e

The Number e

e is an irrational constant ≈ 2.71828... It is the base of natural logarithms.

ln x = logₑ x

Key Properties of ln

  • ln 1 = 0
  • ln e = 1
  • ln eˣ = x
  • e^(ln x) = x
  • ln(AB) = ln A + ln B
  • ln(A/B) = ln A − ln B
  • ln(Aⁿ) = n ln A

Applications of e

  • Compound interest: A = Pe^(rt)
  • Population growth
  • Radioactive decay
  • Newton's Law of Cooling
  • Circuit analysis

📝 Example

Solve: e^(2x) = 15

ln(e^(2x)) = ln 15

2x = ln 15 = 2.708

x = 1.354

🔢 Arithmetic Sequences

Definition

An arithmetic sequence is a sequence where each term is obtained by adding a fixed number (common difference d) to the previous term.

Examples: 2, 5, 8, 11, 14, ... (d = 3)  |  20, 15, 10, 5, ... (d = −5)

nth Term Formula

Tₙ = a + (n−1)d

a = first term, d = common difference, n = term number

Example 1:

Find the 15th term of 3, 7, 11, 15, ...

a = 3, d = 4, n = 15

T₁₅ = 3 + (15−1)×4 = 3 + 56 = 59

Example 2:

Which term of 5, 8, 11, ... is 50?

50 = 5 + (n−1)×3 → 45 = 3(n−1) → n−1 = 15 → n = 16

➕ Arithmetic Series (Sum)

Sum Formula

Sₙ = n/2 [2a + (n−1)d]  or  Sₙ = n/2 (a + l)

where l = last term = a + (n−1)d

Example 1:

Find the sum of the first 20 terms of 4, 7, 10, 13, ...

a = 4, d = 3, n = 20

S₂₀ = 20/2 [2(4) + (20−1)(3)] = 10[8 + 57] = 10 × 65 = 650

Example 2 — Real life:

A worker earns Ksh 8,000 in January and gets a Ksh 500 raise each month. Total earnings in 12 months?

a = 8000, d = 500, n = 12

S₁₂ = 12/2[2(8000) + 11(500)] = 6[16000 + 5500] = 6 × 21500 = Ksh 129,000

📐 Geometric Sequences

Definition

A geometric sequence has a constant ratio r between consecutive terms.

Common ratio: r = T₂/T₁ = T₃/T₂ (any term divided by previous term)
Examples: 2, 6, 18, 54, ... (r = 3)  |  64, 32, 16, 8, ... (r = 0.5)

nth Term Formula

Tₙ = arⁿ⁻¹

a = first term, r = common ratio, n = term number

Example 1:

Find the 6th term of 3, 6, 12, 24, ...

a = 3, r = 2, n = 6

T₆ = 3 × 2⁵ = 3 × 32 = 96

Example 2:

In a GP, T₂ = 6 and T₅ = 48. Find a and r.

ar = 6 ... (i)    ar⁴ = 48 ... (ii)

(ii)÷(i): r³ = 8 → r = 2 → a = 3

✖️ Geometric Series (Sum)

Sum Formula

Sₙ = a(rⁿ − 1)/(r − 1)  when r ≠ 1, r > 1
Sₙ = a(1 − rⁿ)/(1 − r)  when r < 1
Example 1:

Find the sum of first 8 terms of 2, 6, 18, ...

a = 2, r = 3, n = 8

S₈ = 2(3⁸ − 1)/(3−1) = 2(6561−1)/2 = 6560

Example 2 — Population growth:

A town has 5,000 people. Population grows by 10% per year. Total population over 5 years?

a = 5000, r = 1.1, n = 5

S₅ = 5000(1.1⁵−1)/(1.1−1) = 5000(0.6105)/0.1 = 30,526

∞ Sum to Infinity

Convergent Geometric Series

When |r| < 1, a geometric series converges (approaches a finite sum as n → ∞).

S∞ = a / (1 − r)   where |r| < 1
⚠️ If |r| ≥ 1, the series diverges (no sum to infinity).
Example 1:

Find S∞ of 16, 8, 4, 2, ...

a = 16, r = 0.5

S∞ = 16/(1−0.5) = 16/0.5 = 32

Example 2 — Recurring decimals:

Express 0.333... as a fraction using GP.

0.333... = 0.3 + 0.03 + 0.003 + ... → a = 0.3, r = 0.1

S∞ = 0.3/(1−0.1) = 0.3/0.9 = 1/3

Identifying the Type

Is 5, 10, 20, 40 an AP or GP?

Differences: 5, 10, 20 — NOT constant → not AP

Ratios: 10/5=2, 20/10=2, 40/20=2 — constant → GP, r=2

💰 Real-Life Applications

Compound Interest (GP)

Example:

Ksh 10,000 invested at 12% compound interest per year. Amount after 5 years?

a = 10,000, r = 1.12, n = 5

T₅ = 10,000 × 1.12⁴ = 10,000 × 1.5735 = Ksh 15,735

Salary Increments (AP)

Example:

Starting salary Ksh 30,000, annual increment Ksh 2,000. Salary in 10th year?

T₁₀ = 30,000 + (10−1)×2,000 = 30,000 + 18,000 = Ksh 48,000

Bouncing Ball (GP)

Example:

A ball dropped from 10 m bounces to 60% of previous height. Total distance travelled?

First drop: 10 m. Each bounce up then down: 10×0.6 + 10×0.6 = 2×10×0.6

Total = 10 + 2×[10×0.6/(1−0.6)] = 10 + 2×15 = 40 m

✏️ Practice Problems

Problem 1

The 5th term of an AP is 21 and the 10th term is 41. Find a, d, and S₁₅.

Solution:

T₅ = a + 4d = 21 ... (i)    T₁₀ = a + 9d = 41 ... (ii)

(ii)−(i): 5d = 20 → d = 4, a = 5

S₁₅ = 15/2[2(5)+14(4)] = 15/2[10+56] = 15/2×66 = 495

Problem 2

A GP has first term 3 and sum to infinity 12. Find r and the 4th term.

Solution:

S∞ = a/(1−r) → 12 = 3/(1−r) → 1−r = 0.25 → r = 0.75

T₄ = 3 × (0.75)³ = 3 × 0.4219 = 1.266

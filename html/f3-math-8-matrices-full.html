Matrices — Definition and Order

A matrix is a rectangular array of numbers arranged in rows and columns. The order is m×n (m rows, n columns).

Examples:
A = [[3, 5], [1, 2]] → Order 2×2 (square matrix)
B = [[4, 1, 0], [2, 3, 5]] → Order 2×3
C = [[7], [2], [9]] → Order 3×1 (column vector)

Addition and Subtraction of Matrices

Only matrices of the same order can be added or subtracted — add/subtract corresponding elements.

Example: A = [[3,1],[4,2]], B = [[5,0],[1,3]]
A + B = [[3+5, 1+0], [4+1, 2+3]] = [[8,1],[5,5]]
A − B = [[3−5, 1−0], [4−1, 2−3]] = [[−2,1],[3,−1]]
Scalar multiplication: 3A = 3×[[3,1],[4,2]] = [[9,3],[12,6]]

Matrix Multiplication

Matrix multiplication is possible when columns of A = rows of B. (A: m×n) × (B: n×p) = C: m×p

For 2×2: C = A × B where c_ij = Σ a_ik × b_kj
c₁₁ = a₁₁b₁₁ + a₁₂b₂₁
Example: A = [[2,3],[1,4]], B = [[1,0],[5,2]]
AB[1,1] = 2×1 + 3×5 = 2+15 = 17
AB[1,2] = 2×0 + 3×2 = 0+6 = 6
AB[2,1] = 1×1 + 4×5 = 1+20 = 21
AB[2,2] = 1×0 + 4×2 = 0+8 = 8
AB = [[17,6],[21,8]]

Determinant and Inverse of a 2×2 Matrix

For A = [[a,b],[c,d]]:
det(A) = ad − bc
If det(A) ≠ 0, the inverse exists:
A⁻¹ = (1/det(A)) × [[d,−b],[−c,a]]

If det(A) = 0, the matrix is singular (no inverse).

Example: A = [[4,3],[2,1]]
det(A) = 4×1 − 3×2 = 4 − 6 = −2
A⁻¹ = (1/−2) × [[1,−3],[−2,4]] = [[−0.5, 1.5],[1, −2]]
Check: A × A⁻¹ = [[1,0],[0,1]] ✔

Solving Simultaneous Equations Using Matrices

The system ax + by = e, cx + dy = f can be written as AX = B where:

A = [[a,b],[c,d]], X = [[x],[y]], B = [[e],[f]]
Solution: X = A⁻¹ × B
Solve: 2x + y = 7, x + 3y = 11
A = [[2,1],[1,3]], B = [[7],[11]]
det(A) = 6 − 1 = 5
A⁻¹ = (1/5)[[3,−1],[−1,2]]
X = A⁻¹B = (1/5)[[3×7+(−1)×11],[−1×7+2×11]] = (1/5)[[10],[15]] = [[2],[3]]
∴ x = 2, y = 3

Matrix Properties — Important Rules

Commutativity (Add)A + B = B + A
Commutativity (Mul)AB ≠ BA in general

Zero matrixA + 0 = A; A×0 = 0

Careful: Matrix multiplication is not commutative — order matters!
AB ≠ BA generally. Always multiply in the correct order.

Summary — Key Formulae

A = [[a,b],[c,d]]
det(A) = ad − bc
A⁻¹ = (1/(ad−bc)) × [[d,−b],[−c,a]]
Solve AX = B using:
X = A⁻¹B
Step-by-step for simultaneous equations:
  1. Write in matrix form: AX = B
  2. Find det(A) = ad − bc (check ≠ 0)
  3. Write A⁻¹ = (1/det) × [[d,−b],[−c,a]]
  4. Multiply: X = A⁻¹B to get x and y

Types of Errors

Absolute Error

Absolute Error = |Measured Value − True Value|

Example: True value = 5.0cm, Measured = 4.8cm → Absolute error = |4.8 − 5.0| = 0.2cm

Relative Error

Relative Error = Absolute Error / True Value

Example: Relative error = 0.2/5.0 = 0.04

Percentage Error

% Error = (Absolute Error / True Value) × 100%

Example: % Error = (0.2/5.0) × 100% = 4%

Maximum Absolute Error of a rounded measurement = half the unit of rounding. E.g. if rounded to nearest cm, max error = 0.5cm

Upper & Lower Bounds

Bounds of a Measurement

When a measurement is given to a certain degree of accuracy, the true value lies within a range:

Lower Bound = value − half unit  |  Upper Bound = value + half unit

Error Propagation in Calculations

Addition & Subtraction

Maximum Error = sum of individual absolute errors

Example: a = 3.5 ± 0.5, b = 2.3 ± 0.5
a + b = 5.8, max error = 0.5 + 0.5 = 1.0
Result: 5.8 ± 1.0

Multiplication & Division

Max Relative Error = sum of individual relative errors

Example: a = 4cm ± 0.5, b = 6cm ± 0.5
Area = 24cm². Rel. error = 0.5/4 + 0.5/6 = 0.125 + 0.083 = 0.208
Max error in area = 0.208 × 24 ≈ 5cm²
Area = 24 ± 5 cm²

📝 Worked Example

Length = 8m (nearest metre), Width = 5m (nearest metre). Find max area.

UB(length) = 8.5m, UB(width) = 5.5m

Max area = 8.5 × 5.5 = 46.75m²

LB(length) = 7.5m, LB(width) = 4.5m

Min area = 7.5 × 4.5 = 33.75m²

Operations in Standard Form

Multiplication: Multiply A values, add powers

(3 × 10⁴) × (2 × 10³) = 6 × 10⁷

Division: Divide A values, subtract powers

(8 × 10⁶) ÷ (4 × 10²) = 2 × 10⁴

Addition/Subtraction in standard form: First convert to the same power of 10, then add/subtract the A values.

Practical Applications

Measurement in Science

  • All measurements have errors
  • Report with appropriate s.f.
  • Errors compound in calculations
  • Use significant figures correctly

Engineering & Construction

  • Tolerance in manufacturing
  • Safety margins in design
  • GPS accuracy limits
  • Survey measurements

📝 Practical Example

A surveyor measures a plot as 25m × 18m, each to the nearest metre. Calculate the maximum possible area.

UB length = 25.5m, UB width = 18.5m

Max area = 25.5 × 18.5 = 471.75m²

Stated area = 25 × 18 = 450m²

Maximum error = 471.75 − 450 = 21.75m²

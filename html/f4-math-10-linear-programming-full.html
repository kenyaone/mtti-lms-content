📐 Introduction to Linear Programming

Key Terms

TermMeaning
Decision variablesThe unknowns we want to find (x and y)
Objective functionWhat we want to maximise or minimise (e.g. profit P = 3x+2y)
ConstraintsThe limiting conditions (inequalities)
Feasible regionThe shaded area satisfying all constraints
Optimal solutionThe best point in the feasible region

General Approach

  1. Define variables (let x = ..., y = ...)
  2. Write the objective function
  3. Write all constraints as inequalities
  4. Draw the feasible region
  5. Test vertices of feasible region in objective function
  6. State the optimal solution
📝 Formulating LP Problems

Worked Example — Formulation

Problem:

A factory makes tables (x) and chairs (y). Each table needs 3 hrs carpentry and 2 hrs finishing. Each chair needs 1 hr carpentry and 2 hrs finishing. Available: 12 hrs carpentry and 16 hrs finishing. Profit: Ksh 400 per table, Ksh 250 per chair. Maximise profit.

Variables: x = tables, y = chairs

Objective: Maximise P = 400x + 250y

Constraints:

  • 3x + y ≤ 12 (carpentry)
  • 2x + 2y ≤ 16, i.e. x + y ≤ 8 (finishing)
  • x ≥ 0, y ≥ 0 (non-negative)
📊 Drawing the Feasible Region

Steps for Drawing

  1. Convert each inequality to an equation (boundary line)
  2. Find x and y intercepts of each line
  3. Draw the lines on a graph
  4. Test a point (usually origin) to determine which side to shade
  5. The feasible region is where ALL shaded regions overlap
For 3x + y ≤ 12:

When x=0: y=12 → point (0,12)

When y=0: x=4 → point (4,0)

Test (0,0): 0 ≤ 12 ✓ → shade side containing origin

For x + y ≤ 8:

Points: (0,8) and (8,0)

Test (0,0): 0 ≤ 8 ✓ → shade side containing origin

🎯 Finding the Optimal Solution

Vertex (Corner Point) Method

The maximum or minimum of the objective function always occurs at a vertex (corner point) of the feasible region.

Finding vertices for our example:

Vertices of feasible region (read from graph or solve simultaneous equations):

O(0,0), A(4,0), B(2,6), C(0,8)

Test P = 400x + 250y at each:

VertexP = 400x+250y
O(0,0)0
A(4,0)1600
B(2,6)800+1500=2300
C(0,8)2000

Maximum profit = Ksh 2,300 at B(2,6): Make 2 tables and 6 chairs.

📉 Minimisation Problems

Minimising Cost

Example:

A diet must contain at least 30g protein and 20g fat. Food A costs Ksh 5/g protein and Ksh 8/g fat. Food B costs Ksh 7/g protein and Ksh 4/g fat.

Minimise: C = 5x + 7y (cost of protein) subject to constraints.

The minimum occurs at a vertex of the feasible region — test all vertices and take the smallest value.

Iso-profit/Iso-cost Line Method

Alternative to vertex method:

  1. Draw the objective function line (P = constant)
  2. Move the line parallel to itself
  3. The last vertex it touches before leaving feasible region is optimal
📊 Integer Programming

When Solutions Must Be Whole Numbers

In many real problems, you can't make 2.5 tables — solutions must be integers.

Integer LP: If the optimal vertex gives non-integer values, test all nearby integer points within the feasible region and pick the best one.
Example:

Optimal solution is x=2.5, y=3.5. Test nearby integers in feasible region:

(2,3), (2,4), (3,3) — calculate P for each and pick maximum.

💡 Real-Life Applications

Business Production Planning

Example — Crop Planting:

A farmer has 10 acres and Ksh 20,000 to plant maize (x) and beans (y). Maize: 2 acres, Ksh 3000/acre. Beans: 1 acre, Ksh 1500/acre. Expected profit: Ksh 4000/acre maize, Ksh 2500/acre beans.

Constraints: 2x+y≤10, 3000x+1500y≤20000, x≥0, y≥0

Maximise: P = 4000x + 2500y

Transport and Logistics

Example:

A truck can carry max 5 tonnes. Small boxes weigh 0.5T, large boxes 1T. Profit: Ksh 200/small box, Ksh 350/large box. At least 2 of each type must be carried.

Maximise: P = 200x + 350y subject to 0.5x+y≤5, x≥2, y≥2

✏️ Practice Problems

Problem 1

Maximise P = 3x+2y subject to: x+y≤4, x+2y≤6, x≥0, y≥0.

Solution:

Vertices: O(0,0), A(4,0), B(2,2), C(0,3)

P values: 0, 12, 10, 6

Maximum P = 12 at (4,0)

Problem 2

Minimise C = 2x+3y subject to: x+y≥6, 2x+y≥8, x≥0, y≥0.

Solution:

Vertices: A(8,0), B(2,4), C(0,8)

C values: 16, 16, 24

Minimum C = 16 at (8,0) or (2,4)

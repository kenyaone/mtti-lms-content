Introduction to Quadratic Equations

Standard Form

ax² + bx + c = 0

where a ≠ 0, and a, b, c are constants.

✅ Quadratic Equations

  • x² + 5x + 6 = 0
  • 2x² - 3x = 0
  • x² - 9 = 0
  • 3x² + 7x - 2 = 0

❌ Not Quadratic

  • x + 5 = 0 (linear)
  • x³ + x = 0 (cubic)
  • 5 = 0 (no variable)
  • 1/x + 3 = 0 (rational)
Key: The highest power of x must be 2 for an equation to be quadratic. The coefficient 'a' of x² must not be zero.

Method 1: Factorisation

Steps for Factorisation

Write the equation in standard form: ax² + bx + c = 0

Find two numbers that multiply to give ac and add to give b

Split the middle term using these two numbers

Factorise by grouping

Set each factor to zero and solve

📝 Worked Example: x² + 5x + 6 = 0

Find two numbers: multiply to 6, add to 5 → 2 and 3

x² + 2x + 3x + 6 = 0

x(x + 2) + 3(x + 2) = 0

(x + 3)(x + 2) = 0

x + 3 = 0  or  x + 2 = 0

x = −3  or  x = −2

📝 Example 2: 2x² − 5x − 3 = 0

ac = 2 × (−3) = −6, b = −5

Two numbers: multiply to −6, add to −5 → 1 and −6

2x² + x − 6x − 3 = 0

x(2x + 1) − 3(2x + 1) = 0

(x − 3)(2x + 1) = 0

x = 3  or  x = −½

Method 2: Completing the Square

Steps

Make coefficient of x² equal to 1 (divide through if needed)

Move constant term to the right side

Add (b/2)² to both sides

Write left side as a perfect square

Take square root of both sides and solve

📝 Example: x² + 6x − 7 = 0

x² + 6x = 7

Add (6/2)² = 9 to both sides:

x² + 6x + 9 = 7 + 9

(x + 3)² = 16

x + 3 = ±4

x = 4 − 3 = 1  or  x = −4 − 3 = −7

Key formula: (x + b/2)² = x² + bx + (b/2)²

Method 3: Quadratic Formula

The Quadratic Formula

x = (−b ± √(b² − 4ac)) / 2a

Used when factorisation is difficult or impossible. Works for all quadratic equations.

📝 Example: 2x² + 3x − 2 = 0

a = 2, b = 3, c = −2

x = (−3 ± √(3² − 4×2×(−2))) / (2×2)

x = (−3 ± √(9 + 16)) / 4

x = (−3 ± √25) / 4

x = (−3 ± 5) / 4

x = (−3 + 5)/4 = ½  or  x = (−3 − 5)/4 = −2

The Discriminant: b² − 4ac

ValueNature of Roots
b² − 4ac > 0Two distinct real roots
b² − 4ac = 0One repeated real root
b² − 4ac < 0No real roots (complex)

Special Cases

Difference of Two Squares

x² − a² = (x+a)(x−a)

Example: x² − 25 = 0

(x + 5)(x − 5) = 0

x = 5 or x = −5

Perfect Square Trinomial

(x + a)² = x² + 2ax + a²

Example: x² + 10x + 25 = 0

(x + 5)² = 0

x = −5 (repeated root)

When b = 0: Pure Quadratic

Example: 3x² − 48 = 0

3x² = 48

x² = 16

x = ±4

When c = 0

Example: x² − 5x = 0

x(x − 5) = 0

x = 0 or x = 5

Word Problems

Problem-Solving Steps

Read carefully and identify the unknown (let x = ...)

Form a quadratic equation from the given information

Solve the equation using appropriate method

Check answers in context (reject negative values where inappropriate)

📝 Example 1: Area Problem

A rectangular garden is 3m longer than it is wide. Its area is 40m². Find its dimensions.

Let width = x, length = x + 3

x(x + 3) = 40

x² + 3x − 40 = 0

(x + 8)(x − 5) = 0

x = 5 (reject x = −8)

Width = 5m, Length = 8m

📝 Example 2: Number Problem

The product of two consecutive even numbers is 48. Find the numbers.

Let numbers = n and n + 2

n(n + 2) = 48 → n² + 2n − 48 = 0

(n + 8)(n − 6) = 0

n = 6 → numbers are 6 and 8

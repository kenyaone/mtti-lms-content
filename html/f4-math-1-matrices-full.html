📐 What is a Matrix?

Definition

A matrix is a rectangular array of numbers arranged in rows and columns, enclosed in brackets.

Key Term — Matrix: An ordered rectangular array of numbers. The order (or size) of a matrix is written as m × n, where m = number of rows and n = number of columns.

Example of a 2×3 matrix (2 rows, 3 columns):

A =
123 456

Each number in a matrix is called an element or entry. Element aij is in row i, column j.

All elements are zero.

➕ Addition & Subtraction of Matrices

Rule

You can only add or subtract matrices of the same order. Add/subtract corresponding elements.

⚠️ Matrices of different orders cannot be added or subtracted.

Possible only when the number of columns in A = number of rows in B.

Row1 × Col1: (1×5)+(2×7) = 5+14 = 19

🔢 Determinant of a 2×2 Matrix

Definition

The determinant is a single number calculated from a square matrix. For a 2×2 matrix:

A =
ab cd

det(A) = |A| = ad − bc

Worked Examples

Example 1:
A =
31 24

det(A) = (3×4) − (1×2) = 12 − 2 = 10

Example 2:
B =
52 31

det(B) = (5×1) − (2×3) = 5 − 6 = −1

Singular Matrix: A matrix with determinant = 0 has no inverse and is called singular.
🔄 Inverse of a 2×2 Matrix

Formula

For matrix A = [a b; c d], the inverse A⁻¹ is:

A⁻¹ = 1/det(A) × [d −b; −c a]

Swap the diagonal elements, change the signs of the off-diagonal elements, then multiply by 1/det(A)

Check: A × A⁻¹ = I (the identity matrix)

Worked Example

Find the inverse of A:

🧮 Solving Simultaneous Equations Using Matrices

Matrix Method

A system of simultaneous equations can be written as AX = B, where A is the coefficient matrix, X is the variable matrix, and B is the constant matrix.

Solution: X = A⁻¹B

Worked Example

Solve: 2x + y = 5 and x + 3y = 10

Step 1: Write in matrix form AX = B:

21 13
x y
=
5 10

Step 2: det(A) = (2×3)−(1×1) = 5

Step 3: A⁻¹ = 1/5 × [3 −1; −1 2]

Step 4: X = A⁻¹B:

x y
= 1/5 ×
3-1 -12
5 10
=
1 3

∴ x = 1, y = 3

Problem 2

Find the determinant and inverse of M = [4 2; 3 1].

Solution:

det(M) = (4×1) − (2×3) = 4 − 6 = −2

M⁻¹ = 1/(−2) × [1 −2; −3 4] = [−0.5 1; 1.5 −2]

Problem 3

Use matrices to solve: 3x + 2y = 12 and x + y = 5.

Solution:

det(A) = (3×1) − (2×1) = 1

A⁻¹ = [1 −2; −1 3]

X = A⁻¹B = [1×12 + (−2)×5; (−1)×12 + 3×5] = [2; 3]

∴ x = 2, y = 3

Sequences and nth Term

A sequence is a list of numbers following a rule. Each number is a term.

nth Term Formula — Examples:
Tₙ = 2n + 1 → 3, 5, 7, 9, 11, … (odd numbers from 3)
Tₙ = n² → 1, 4, 9, 16, 25, … (squares)
Tₙ = 3ⁿ → 3, 9, 27, 81, … (powers of 3)
Tₙ = (−2)ⁿ⁻¹ → 1, −2, 4, −8, … (alternating)

Arithmetic Progression (AP)

An AP has a constant difference d between consecutive terms.

nth term: Tₙ = a + (n−1)d
Sum of n terms: Sₙ = n/2 × [2a + (n−1)d]  or  n/2 × (a + l)
a = first term, d = common difference, l = last term
Example: AP: 5, 8, 11, 14, … Find T₁₅ and S₂₀.
a = 5, d = 3
T₁₅ = 5 + 14×3 = 5 + 42 = 47
S₂₀ = 20/2 × [2(5) + 19×3] = 10 × [10 + 57] = 10 × 67 = 670
Find n: Sum of AP = 108; a=3, d=3. Find n.
Sₙ = n/2[6 + (n−1)3] = n/2[3+3n] = 3n(n+1)/2 = 108
n(n+1) = 72 → n² + n − 72 = 0 → (n+9)(n−8) = 0 → n = 8

Geometric Progression (GP)

A GP has a constant ratio r between consecutive terms.

nth term: Tₙ = arⁿ⁻¹
Sum (r≠1): Sₙ = a(rⁿ−1)/(r−1) if r>1  or  a(1−rⁿ)/(1−r) if r<1
a = first term, r = common ratio
Example: GP: 2, 6, 18, 54, … Find T₈ and S₅.
a = 2, r = 3
T₈ = 2 × 3⁷ = 2 × 2187 = 4 374
S₅ = 2(3⁵−1)/(3−1) = 2(243−1)/2 = 242
Find r: GP: 4, ?, ?, 32. Find r.
T₄ = ar³ = 32; a=4 → 4r³=32 → r³=8 → r = 2
Sequence: 4, 8, 16, 32

Useful identities:
  • If a, b, c are in AP: 2b = a + c
  • If a, b, c are in GP: b² = ac
  • Sum of n natural numbers = n(n+1)/2

Real-Life Applications

AP Application: A worker earns Ksh 8 000 in the first month and gets a Ksh 500 raise each month.
a = 8000, d = 500
Salary in month 12: T₁₂ = 8000 + 11×500 = Ksh 13 500
Total earnings over 12 months: S₁₂ = 12/2 × (8000+13500) = 6 × 21500 = Ksh 129 000
GP Application: A bacterial culture doubles every hour. Start: 500 bacteria.
a = 500, r = 2
After 8 hours: T₉ = 500 × 2⁸ = 500 × 256 = 128 000
Compound Interest as GP: Ksh P at r% p.a. → amounts form a GP with ratio (1 + r/100)

Summary — All Formulae

AP
Tₙ = a + (n−1)d
Sₙ = n/2[2a + (n−1)d] = n/2(a+l)
GP
Tₙ = arⁿ⁻¹
Sₙ = a(rⁿ−1)/(r−1) for r > 1  |  a(1−rⁿ)/(1−r) for r < 1
Key steps for any sequence problem:
  1. Identify if AP (constant difference) or GP (constant ratio)
  2. Find a, d (AP) or a, r (GP) from given terms
  3. Apply appropriate nth term or sum formula
  4. For "find n" problems, set up equation and solve

📈 Rate of Change & Gradient

What is Differentiation?

Differentiation is the process of finding the rate of change of a function. The result is called the derivative or gradient function.

From first principles — for y = x²:

  • Take point P(x, x²) and nearby point Q(x+h, (x+h)²)
  • Gradient of PQ = [(x+h)² − x²] / h = (2xh + h²) / h = 2x + h
  • As h → 0: gradient = 2x
If y = xⁿ, then dy/dx = nxⁿ⁻¹

This is the Power Rule — the most important rule in differentiation.

Delta Notation

Δx (delta x) means a small change in x. Δy means a small change in y.

dy/dx = lim(Δx→0) [Δy/Δx]

🔢 Differentiating Polynomials

The Power Rule

y = xⁿ → dy/dx = nxⁿ⁻¹
y = axⁿ → dy/dx = naxⁿ⁻¹
y = c (constant) → dy/dx = 0

📐 Tangents and Normals

Tangent to a Curve

The tangent at a point on a curve has the same gradient as the curve at that point.

  1. Find dy/dx (the gradient function)
  2. Substitute the x-value to get the gradient m
  3. Use y − y₁ = m(x − x₁) for the equation
Example: Find the tangent to y = x² − 3x + 2 at x = 2.
dy/dx = 2x − 3. At x=2: m = 2(2)−3 = 1
y at x=2: y = 4−6+2 = 0. Point: (2, 0)
Tangent: y − 0 = 1(x − 2) → y = x − 2

Normal to a Curve

The normal is perpendicular to the tangent. If gradient of tangent = m, then gradient of normal = −1/m
Example (continued): Normal at (2,0): gradient = −1/1 = −1
Normal: y − 0 = −1(x − 2) → y = −x + 2

⛰️ Stationary Points & Curve Sketching

Stationary Points

A stationary point occurs where dy/dx = 0 (the gradient is zero — the curve is momentarily flat).

Types of stationary points:

  • Maximum: d²y/dx² < 0 (curve turns from up to down)
  • Minimum: d²y/dx² > 0 (curve turns from down to up)
  • Point of inflection: d²y/dx² = 0

Worked Example

Find the stationary points of y = x³ − 3x² − 9x + 5

Step 1: dy/dx = 3x² − 6x − 9
Step 2: Set to 0: 3x² − 6x − 9 = 0
x² − 2x − 3 = 0 → (x−3)(x+1) = 0
x = 3 or x = −1

Step 3: d²y/dx² = 6x − 6
At x=3: d²y/dx² = 12 > 0 → Minimum
At x=−1: d²y/dx² = −12 < 0 → Maximum

Points: Minimum at (3, −22), Maximum at (−1, 10)

🚗 Applications: Kinematics

Distance, Velocity, Acceleration

s = displacement (distance function)
v = velocity = ds/dt (1st derivative of s)
a = acceleration = dv/dt = d²s/dt² (2nd derivative of s)

Worked Examples

Example 1: A particle moves with s = 2t³ − 9t² + 12t. Find:

(a) Velocity: v = ds/dt = 6t² − 18t + 12

(b) Acceleration: a = dv/dt = 12t − 18

(c) When is velocity zero?
6t² − 18t + 12 = 0 → t² − 3t + 2 = 0
(t−1)(t−2) = 0 → t = 1 or t = 2

(d) Acceleration at t = 3:
a = 12(3) − 18 = 18 m/s²
Example 2 (Kenyan context): The height h of a ball thrown up is h = 20t − 5t².
Maximum height when dh/dt = 0: 20 − 10t = 0 → t = 2 s
Max height = 20(2) − 5(4) = 20 m

🏭 Applications: Maxima & Minima

Optimisation Problems

Many real-world problems involve finding maximum or minimum values. Use differentiation to solve them.

  1. Write an equation for the quantity to optimise
  2. Differentiate and set equal to zero
  3. Solve for the variable
  4. Verify it's a max/min using second derivative
Example: A farmer in Eldoret has 40 m of fencing to enclose a rectangular garden. Find the dimensions for maximum area.

Let width = x. Then length = (40−2x)/2 = 20−x
Area A = x(20−x) = 20x − x²
dA/dx = 20 − 2x = 0 → x = 10
d²A/dx² = −2 < 0 (maximum confirmed)

Maximum area: A = 10 × 10 = 100 m² (a square!)
